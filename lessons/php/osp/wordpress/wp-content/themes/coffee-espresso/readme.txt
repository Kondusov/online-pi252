== Description ==

Coffee Espresso Studio theme is a thoughtfully crafted solution designed for coffee shop websites, café businesses, espresso bars, coffee roasteries, bakery cafes, and small food businesses that want a clean and structured online presence. Built with a focus on usability, it offers a flexible layout that supports menu display, beverage listings, product showcasing, and brand storytelling through visuals. The theme integrates smoothly with Gutenberg, making content editing simple and efficient. It includes dedicated sections for featured coffee drinks, espresso menus, pricing tables, gallery layouts, customer testimonials, and daily specials, helping businesses present their offerings clearly. With optimized performance and lightweight coding, the Coffee Espresso Studio theme ensures faster loading and a smooth browsing experience across devices. Key features include responsive design, SEO-friendly structure, translation readiness, blog support, and easy customization of colors and typography. It also supports essential features like contact forms (compatible with Contact Form 7 plugin) for inquiries and reservations. This theme is ideal for coffee shops, café websites, artisan coffee brands, bakery stores, and modern espresso businesses looking to build a refined and engaging digital presence while improving visibility for searches like coffee shop near me and espresso café.

== Installation ==

1. In your admin panel, go to Appearance > Themes and click the Add New button.
2. Click Upload Theme and Choose File, then select the theme's .zip file. Click Install Now.
3. Click Activate to use your new theme right away.
